Get Started
1

Get your Kite API credentials

Create or open your app at developers.kite.trade/apps to get your API key and secret.

2

Connect via your MCP client

Add the server to Claude Code, VS Code, or any MCP-compatible client:

# Claude Code claude mcp add kite-fly --transport streamable-http \ https://kite-mcp-server.fly.dev/mcp
3

Authenticate

Call the login tool with your api_key and api_secret. You will be redirected to Kite to authorize. Your session persists across server restarts.
